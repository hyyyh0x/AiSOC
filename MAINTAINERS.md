# AiSOC Maintainers

Maintainers review and merge pull requests, triage issues, cut releases, and
steward the roadmap. See [`GOVERNANCE.md`](GOVERNANCE.md) for roles, the
decision-making process, and how to become a maintainer.

## How to reach the maintainers

- **General / roadmap:** open a GitHub issue or discussion.
- **Security:** follow [`SECURITY.md`](SECURITY.md) — never a public issue.

## Becoming a maintainer

The project actively wants to grow this list. New maintainers are nominated by
an existing maintainer via a PR editing this file, under the "substantive
change" review rules in `GOVERNANCE.md`. Sustained, high-quality contributions
and reviews are the path in.
